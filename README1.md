# ProMed
A mobile app built using Flutter / Django Rest that sells licenses to watch medical courses in Arabic.
This repo contains the backend of the app.

## Requirements
- Docker
- Docker Compose

## Development
- Clone project
- Run `docker-compose up -build`
- Visit localhost:8000
